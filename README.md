# **Agentic AI @** **`CrewAI`** **+** **`Google Gemini`**

This repository implements an agentic AI workflow using [CrewAI](https://github.com/crewAIInc/crewAI) with [Google Gemini](https://ai.google.dev/) models to orchestrate autonomous multi-agent execution for research and content generation. The architecture is modularized across `agents.py`, `tasks.py`, `tools.py`, and `crew.py`, where agents encapsulate reasoning behavior, tasks define execution objectives, tools enable external knowledge acquisition, and the Crew runtime coordinates execution. Configuration and environment initialization are managed through [python-dotenv](https://github.com/theskumar/python-dotenv).

Agent execution is enabled through CrewAI’s [LiteLLM](https://github.com/BerriAI/litellm) abstraction layer, allowing seamless invocation of `gemini/gemini-3.5-flash` as the underlying LLM. Specialized agents operate with assigned roles, memory, delegation capabilities, and contextual backstories, enabling controlled separation of concerns across the workflow. External grounding is introduced through [CrewAI Tools](https://github.com/crewAIInc/crewAI-tools) using [Serper](https://serper.dev/) search integration, allowing agents to retrieve and incorporate live information into intermediate reasoning stages.

At runtime, CrewAI composes agents and tasks into a unified `Crew` object executed using `Process.sequential`, enforcing ordered context propagation between execution stages. Research outputs generated by the researcher agent become structured inputs for the writer agent, which synthesizes the final markdown artifact (`new-blog-post.md`). The implementation provides a lightweight foundation for agent orchestration, tool-augmented reasoning, autonomous knowledge synthesis, and extensible multi-agent AI systems.
